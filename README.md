###### yolo 가상환경 만들기
``` bash
uname -a
wget https://github.com/Archiconda/build-tools/releases/download/0.2.3/Archiconda3-0.2.3-Linux-aarch64.sh
sudo chmod 755 Archiconda3-0.2.3-Linux-aarch64.sh
````

#### 용어정리 

``` bash
conda : Conda는 파이썬과 라이브러리들을 관리하고 설치할 수 있는 도구로, 가상 환경을 만들어 서로 다른 프로젝트의 패키지 버전을 분리해 관리할 수 있게 해줘요. YOLO 설치 시 필요한 라이브러리들도 쉽게 설치하고 충돌 없이 사용할 수 있습니다.
archiconda : Archiconda는 Arch Linux에서 Conda를 쉽게 설치하고 사용할 수 있게 해주는 버전이에요. Conda로 파이썬 환경을 설정하고 패키지 관리를 할 수 있게 도와줘요.
